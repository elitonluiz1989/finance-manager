# Finance Manager

**Finance Manager** is a robust, cross-platform application designed to give you total control over your financial life. Built using **Kotlin Multiplatform (KMP)**, it provides a seamless experience across mobile, desktop, and web environments.

Unlike standard expense trackers, Finance Manager features a dedicated system to manage **third-party loans**, ensuring you never lose track of money you have lent to others.

## 🚀 Supported Platforms

Thanks to Kotlin Multiplatform, this project targets:
* **Android:** Native mobile experience.
* **Desktop:** Native binaries for **Windows** (.msi) and **Linux** (.deb).
* **Web:** High-performance web version using Kotlin/Wasm or Kotlin/JS.

## ✨ Key Features

* **Expense Tracking:** Categorize and monitor your personal spending habits.
* **Loan Management:** Register and track money lent to individuals, including due dates and repayment status.
* **Unified Balance:** View your true net worth by combining liquid cash with outstanding receivables.
* **Shared Logic:** 100% of the business logic, validation, and data persistence are shared across all platforms using a single Kotlin codebase.

## 🛠 Tech Stack

* **Language:** Kotlin
* **Framework:** Kotlin Multiplatform (KMP)
* **UI:** Compose Multiplatform (Desktop/Android/Web)
* **Networking:** Ktor
* **Local Database:** SQLDelight
* **Dependency Injection:** Koin
